Main File Structure

All the file are well organized, its so easy to work with the template.
1. Unzip the files.
2. Open " kotha-pro-html " folder.
You will find the two folder one is kotha-pro-with-scss & other is kotha-pro--normal-css. You open your desire one and open.
You will find the the index.html file on the root that will be edited.
Also you will see 1 folded named assets. In assets folder you cas see 4 directories css, js, fonts and images
In the first folder Content you will find style.css and other styling files the all the other files expect the htmls.
Js folder contain only the jquery library. Fonts folder contain essential font files for fonts.
